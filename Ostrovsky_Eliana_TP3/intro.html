Trabajo Práctico 3: Redes Neuronales

En este trabajo se desarrolla una red neuronal capaz de clasificar caracteres japoneses a partir de imágenes en escala de grises de 28x28 píxeles. El conjunto de datos contiene 5.000 muestras pertenecientes a 49 clases distintas.

Se implementó una red neuronal desde cero utilizando Python, entrenando modelos con diferentes configuraciones y aplicando técnicas como backpropagation, optimización con ADAM, regularización mediante dropout y early stopping, entre otras. Posteriormente, se comparó el rendimiento de estas implementaciones propias con modelos equivalentes desarrollados en PyTorch, un framework ampliamente utilizado en el campo del aprendizaje profundo.

A lo largo del trabajo se realizaron múltiples experimentos para analizar el impacto de cada componente del entrenamiento, incluyendo estudios de arquitectura, tuning de hiperparámetros y evaluación en distintos subconjuntos de datos. El modelo con mejor desempeño fue finalmente utilizado para generar predicciones sobre un conjunto nuevo de imágenes.
